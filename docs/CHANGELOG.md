# 更新日志

所有项目的显著变更都将记录在此文件中。

## [3.0.0] - 2025-01-05

### 新增

#### 消融测试系统 (Ablation Study)
- **ARPM总开关**: 一键切换纯LLM与增强模式
- **BM25+开关**: 控制关键词检索组件
- **CoT重排序开关**: 控制思维链分析
- **时间衰减开关**: 控制记忆时间权重
- **关键词提升开关**: 控制动态加权
- 保存配置时显示组件状态提示

#### 消息反馈机制
- **点赞功能**: 记录用户偏好用于学习
- **差评功能**: 标记不喜欢的回复风格
- **举报功能**: 
  - 6种举报原因选择
  - 不合规内容自动归档
  - 自动重新生成安全回复
  - 显示"已重新生成"标记

#### 聊天记录管理
- **单条删除**: 鼠标悬停显示删除按钮，带确认对话框
- **清空记录**: 底部"清空记录"按钮
- **二次确认弹窗**: 屏幕居中显示确认信息
- **动画效果**: 删除时淡出动画

#### 知识库管理增强
- **RAG检索功能**: 使用语义向量搜索知识片段
- **统计面板**: 按类型显示片段数量
- **来源分布**: 显示各来源文档统计
- **分页控制**: 支持20/50/100/200条/页
- **性能优化**: 支持4537+片段流畅管理

#### 自动切分重构
- **保留上下文**: 不再删除历史消息
- **ID标记防重**: 使用Set记录已切分消息
- **防抖机制**: 60秒冷却时间
- **控制台日志**: 详细的切分状态输出

#### 系统诊断
- **8项健康检查**: 向量库、元数据、索引、模型等
- **自动修复**: 一键修复常见问题
- **可视化报告**: 清晰的健康状态显示

#### 其他改进
- 思维链自动隐藏 (支持多种格式)
- 会话隔离机制 (防止消息混淆)
- 增强的错误处理和日志记录

### 优化
- 知识库管理界面重构
- 消息渲染性能优化
- API请求错误处理增强
- 前端状态管理改进

### 修复
- 修复大数量片段卡顿问题
- 修复自动切分重复触发问题
- 修复消息时间戳不一致问题
- 修复切换会话后状态混淆问题

## 版本说明

### 版本号规则
- **主版本号**: 重大架构变更
- **次版本号**: 功能新增
- **修订号**: Bug修复和优化

### 兼容性
- v3.0 数据格式与 v2.0 兼容
- 知识库数据自动迁移
- 配置文件需要重新设置

### 升级指南

**从 v2.0 升级到 v3.0**:
1. 备份 data/ 目录
2. 拉取新版本代码
3. 更新依赖: `pip install -r requirements.txt`
4. 启动服务
5. 重新配置API设置
6. 知识库数据自动保留

**注意事项**:
- v3.0 新增了许多配置项，建议查看设置面板
- 消融测试功能默认全部开启
- 消息反馈功能需要用户主动使用
