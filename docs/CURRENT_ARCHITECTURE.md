# 当前工程状态

## 目录约定

- `code/arpm-app`
  代码仓主体，只放源码、脚本、文档和依赖定义。
- `runtime/arpm-app`
  运行时数据目录，包含 `data`、`logs` 等可变内容。
- `assets/models`
  模型目录。
- `env/arpm-venv`
  本地虚拟环境目录。
- `research/thesis`
  论文与研究材料目录。
- `release`
  打包归档目录。

## 当前真实链路

1. 用户消息进入 `/api/chat`
2. 检索器分别召回：
   - 全局知识库
   - 当前会话对话历史
3. 知识库结果和对话历史结果分别独立计算权重
4. 生成器生成回复
5. 生成完成后执行对话原子写入：
   - 写入会话 JSON
   - 写入当前会话的 chat 向量索引

## 关键约束

- 知识库为全局共享索引
- 对话历史为按 `session_id` 隔离索引
- 原子写入不参与知识库权重计算
- 双时态权重只作用于检索结果重排

## 当前前端范围

当前主前端只对齐以下真实能力：

- 会话切换与历史加载
- 聊天发送、停止、重生成、删除轮次
- 当前会话清空 / 当前会话索引清空
- 知识库上传、浏览、删除、清空
- 系统诊断、ARPM 组件报告、在线块删除

## 已移除/不再作为当前实现依据

- `backend/web/static/js/kb-manager.js`
  旧知识库管理脚本，依赖已不存在的历史接口，已移除。

## 启动方式

- 使用 `code/arpm-app/start.bat`
- 运行前要求：
  - 模型位于 `assets/models`
  - 虚拟环境位于 `env/arpm-venv`
  - 运行数据位于 `runtime/arpm-app`
