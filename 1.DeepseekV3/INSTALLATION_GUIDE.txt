【English】
==========

1. REQUIREMENTS
---------------
- Python 3.10+
- Windows/Linux/Mac
- DeepSeek API Key (https://platform.deepseek.com/)

2. INSTALLATION
---------------

Step 1: Clone Repository
~~~~~~~~~~~~~~~~~~~~~~~~
git clone https://github.com/Spirtxiaoqi7/ARPM.git
cd ARPM

Step 2: Create Virtual Environment
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
python -m venv venv

# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate

Step 3: Install Dependencies
~~~~~~~~~~~~~~~~~~~~~~~~~~~~
pip install -r requirements.txt

Step 4: Configure API Key
~~~~~~~~~~~~~~~~~~~~~~~~~
Edit .env file:
DEEPSEEK_API_KEY=your_api_key_here

Step 5: Download Model (Auto or Manual)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
# Auto: Model downloads on first run
# Manual: See models/README.md

Step 6: Run
~~~~~~~~~~~
# Option 1: Windows one-click
RUN.bat

# Option 2: Manual
cd api && uvicorn rag_api:app --port 8003      # Terminal 1
cd api && uvicorn deepseek_api:app --port 8004 # Terminal 2
python chat_loop.py                            # Terminal 3

3. CONFIGURATION
----------------

config.json:
{
    "character_name": "Your Character",
    "character_background": "Character description..."
}

4. ADD DOCUMENTS
----------------
python chunker.py file.txt --add

5. API ENDPOINTS
----------------
RAG:       http://127.0.0.1:8003
DeepSeek:  http://127.0.0.1:8004
