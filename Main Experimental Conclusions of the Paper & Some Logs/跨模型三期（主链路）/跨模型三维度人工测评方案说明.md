# 跨模型人工盲审测评方案说明

## 1. 实验目的

本实验用于对跨模型生成语料进行人工评价，重点考察模型回复在多轮对话场景中的合理性、语气一致性与人格连续性。系统将不同来源的对话轮次统一匿名化为 A-Z 模型标签，由评审者在不知道真实模型名称与来源条件的情况下进行评分，以支持论文中对跨模型表现差异的定量描述。

## 2. 数据来源与样本构成

系统当前支持三类数据：

1. `metadata.json`：以对话轮次为单位的语料数据，包含用户输入、模型回复、轮次编号、时间戳等字段。
2. `思维链+对话.log`：JSONL 日志数据，每行对应一次模型生成记录，包含轮次、生成前分析文本、最终回复与协议信息。
3. `对话内容.log`：对话轮次日志，每行对应一次用户输入与模型回复，包含轮次编号、物理时间、用户输入与模型回复等字段。

为降低数据格式差异带来的偏差，系统会将三类数据统一转换为标准评审样本：匿名模型标签、轮次编号、时间戳、用户提问、生成前分析文本、模型最终回复、原始抽取块。对于思维链日志样本，由于原始文件未提供独立的 `user_input` 字段，系统会从分析文本中抽取用户提问线索，并同时展示完整分析文本与最终回复；对于对话内容日志，系统直接抽取 `user_input` 与 `assistant_reply` 字段，便于评审者按同一标准判断该轮回复质量。

## 3. 匿名化与盲审机制

系统不在评审界面展示真实模型名称、模型来源、实验组名称或文件来源。不同模型或数据来源在进入评审池后会被统一映射为 A-Z 标签。评审者只能看到匿名标签、轮次编号、时间戳以及待评审文本。

样本抽取采用随机方式。每次点击“开始测评”或提交评分后，系统从当前样本池中随机抽取一个轮次展示。该机制允许无限评分，即同一轮次可能被多次抽中，从而可用于增加重复测量数量，后续可根据导出的评分结果计算均值、方差或一致性指标。

## 4. 评分维度与量化规则

每个样本从三个维度进行人工评分：

1. 合理性评估：判断回复内容是否符合上下文、事实与任务目标。
2. 语气一致性：判断回复语气是否与既定角色、对话氛围和上下文风格一致。
3. 人格连续性：判断模型是否保持稳定的人格设定、记忆线索和表达习惯。

每个维度采用五级等级量表：

| 等级 | 含义 | 分值 |
|---|---|---:|
| A | 合理 | 1.0 |
| B | 相对合理 | 0.8 |
| C | 不太合理 | 0.5 |
| D | 不合理 | 0.2 |
| E | 极其不合理 | 0.0 |

系统在右侧实时显示每个匿名模型在三个维度上的平均分，以及该模型已评分样本数。

## 5. 客观性控制

本方案通过以下方式提升实验客观性：

1. 模型匿名化：评审者无法直接看到模型真实名称，降低品牌印象、模型预期和先验偏好的影响。
2. 随机抽取：评审顺序不由研究者手动指定，减少选择性呈现造成的顺序偏差。
3. 统一量表：所有模型使用相同的三维度五级评分标准，保证不同模型之间具有可比性。
4. 来源兼容：不同格式的数据被统一为同一评审结构，减少文件格式差异对评分流程的影响。
5. 上下文透明：日志样本同时展示可提取的用户提问、生成前分析与最终回复，使评审者能够基于完整上下文判断合理性、语气一致性与人格连续性。
6. 可重复导出：评分结果以结构化 JSON 导出，包含匿名模型、轮次、时间、维度分数和评分时间，便于复核与统计分析。

需要说明的是，人工评分仍然包含主观判断成分。若用于正式论文，建议引入多名评审者，并报告评分者间一致性，例如 Cohen's kappa、Krippendorff's alpha 或组内相关系数。若只有单名评审者，则应在论文中说明该实验属于人工评估辅助分析，结论以描述性统计为主。

## 6. 使用流程

1. 启动本地 Web 服务并打开 `http://127.0.0.1:8000/index.html`。
2. 页面自动读取同目录下的 `metadata.json`、`思维链+对话.log` 与 `对话内容.log`。
3. 点击“开始测评”，系统随机抽取一个匿名样本。
4. 评审者分别对合理性、语气一致性、人格连续性进行 A-E 评分。
5. 点击“提交并抽下一轮”，系统保存本次评分并继续随机抽取。
6. 右侧面板实时显示 A-Z 匿名模型的平均得分与样本数。
7. 评分完成后点击“导出结果”，得到可用于论文统计分析的 JSON 文件。

## 7. 论文表述建议

论文中可将该方法描述为“随机化匿名人工评估”。核心表述为：研究者将跨模型生成语料转换为统一评审样本，对模型身份进行 A-Z 匿名标记，并通过 Web 端随机呈现给评审者。评审者基于预定义量表从合理性、语气一致性和人格连续性三个维度进行独立评分。系统实时累计各匿名模型的维度均值，并导出结构化评分记录用于后续统计分析。该流程通过匿名化、随机化、统一量表和结构化导出降低主观偏差，提高跨模型比较的透明度与可复核性。
